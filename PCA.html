1 Introducci√≥n

¬øQu√© es?

Es una t√©cnica multivariada de ordenaci√≥n de interdependencia con enfoque descriptivo. La para interdepencia hace referencia a que en la matriz ninguna de las variables utilizadas figura como dependiente principal de las otras. En este an√°lisis las componentes principales son nuevas variables creadas a partir de un conjunto de variables originales como combinaciones lineales de las mismas.

La estructuraci√≥n de la t√©cnica se debe a Hotelling (1933), aunque sus or√≠genes se encuentran en los ajustes ortogonales por m√≠nimos cuadrados introducidos por K. Pearson (1901).

¬øCu√°les son los objetivos del an√°lisis?

  • Representar √≥ptimamente en un espacio de dimensi√≥n peque√±a, observaciones de un espacio p-dimensional con la m√≠nima perdida de la informaci√≥n.
  • Identificar posibles variables latentes o no observadas, que est√°n generando la variabilidad de los datos.
  • Transformar las variables originales, en general correladas, en nuevas variables incorreladas, facilitando la interpretaci√≥n de los datos.

¬øQu√© se requiere para aplicar el an√°lisis a una matriz de datos b√°sica?

Para poder aplicar el an√°lisis de importante que:

  1. El conjunto de datos p-dimensional se encuentre correlacionado.
  2. En la matriz de datos b√°sica ninguna de las variables destaque como variable dependiente principal.
  3. Normalidad p-variante solo en el enfoque inferencial.

¬øPor qu√© es importante esta t√©cnica en los contextos bi√≥logicos?

Los conjuntos de datos biol√≥gicos suelen tener un gran n√∫mero de variables. Los componentes principales permiten reducir la dimensionalidad sin perder mucha informaci√≥n, facilitando la identificaci√≥n de patrones y relaciones subyacentes entre las variables que podr√≠an ser dif√≠ciles de ver en el conjunto de datos original, visualizaci√≥n de la informaci√≥n y el an√°lisis posterior con otras t√©cnicas.

6.2 PASO 2. Obtenci√≥n de las componentes

Dos son las formas m√°s comunes de generar las componentes principales:

  1. A partir de la matriz de varianzas y covarianzas: Si las diferencias entre las varianzas de las variables son informativas y queremos tenerlas en cuenta
  2. A trav√©s de la matriz de correlaci√≥n. Cuando las variables originales est√°n en distintas unidades y si hay diferencia entre las varianzas de las variables originales y no queremos que hayan sesgos en el an√°lisis.

En cualquiera de los dos casos para obtener las componentes principales se requiere calcular los valores y vectores propios de tal matriz (Matriz S de varianzas - Covarianzas √≥ la matriz R de correlaci√≥n).

Definici√≥n de componente.

Cada componente principal denotada con \(W_i\) es una combinaci√≥n lineal de las variables originales, donde \(a_i\) son el vector propio \(i\) y corresponde al peso de la variable \(x_i\) de la matriz de datos en la componente principal; obteniendose as√≠ tantas componentes como p variables tenga nuestra matriz.

Sea la componente \(i\), \[ W_i=X*a_i \]

La funci√≥n "PCA()" permite obtiener las componentes a partir de la matriz de correlaciones, el argumento "scale.unit" es un argumento l√≥gico que se le puede a√±adir a la l√≠nea de c√≥digo para indicar la estandarizaci√≥n de las variables cuando estas se encuentras en unidades de medici√≥n distintas y el argumento "ncp" es el n√∫mero de componentes a calcular.

6.5.2 Circulo de correlaciones

  3. Las variables correlacionadas entre si forman angulos menore a 90¬∞ √≥ mayores a 90¬∞, las no correlacionadas forman √°ngulos de 90¬∞.
  4. La magnitud del vector mide la calidad de las variables en el c√≠rculo. Variables con mayor magnitud; es decir, que est√°n lejos del origen est√°n bien representadas, lo que implican que tiene mayor capacidad informativa.
